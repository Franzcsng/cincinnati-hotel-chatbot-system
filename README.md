# The Cincinnati Hotel

"Stay where the city began." A boutique hotel website (Downtown Cincinnati /
Over-the-Rhine) with separate guest and admin experiences, plus an
admin-uploaded PDF knowledge base for an AI assistant.

- **Frontend:** `frontend/` — React 19 + Vite
- **Backend:** `server/` — Node.js + Express

## Documentation

Full project documentation lives in [`docs/`](./docs/README.md), including
setup instructions, architecture, API reference, environment variables, and
the brand/design system. Keep it updated as the project evolves.

## Quick start

```bash
cd frontend && npm install && npm run dev   # http://localhost:5173
cd server && npm install && npm run dev     # http://localhost:4000
```

See [docs/setup.md](./docs/setup.md) for full setup details.
